#  Catalogue Integration Assistant
A Production oriented Retrieval-Augmented Generation(RAG) assistant for marketplace catalog integration, built with Amazon Bedrock

## Overview
This project demonstrates a production-ready retrieval-augmented generation (RAG) system for answering questions about marketplace feed specifications, error codes, 
and seller onboarding — grounded in source documents with citations.

Phase 1: Local RAG with Bedrock Titan embeddings and Claude
Phase 2: AWS-native with Knowledge Base + OpenSearch Serverless
Phase 3: Serverless API (Lambda + API Gateway)
Phase 4: Production hardening (Guardrails, evaluation, monitoring)
Phase 5: Infrastructure as Code (CDK/Terraform)

## Technologies

- Python 3.10+
- Amazon Bedrock (Titan embeddings, Claude)
- NumPy
- Boto3
